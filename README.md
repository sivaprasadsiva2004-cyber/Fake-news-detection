# Fake-news-detection
Classify global news articles as "Real" or "Fake" using NLP.  Algorithms: Passive Aggressive Classifier, Naive Bayes, Logistic Regression. Preprocessing: Lemmatization, Stopword removal TF-IDF Vectorization. Key Achievement: Developed a "Publisher-Strip" regex to remove data leakage ensuring the model learns actual context rather than source names.
